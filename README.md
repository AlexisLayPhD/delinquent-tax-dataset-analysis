# Delinquent Tax Dataset Analysis

Exploratory data analysis of delinquent tax datasets using Excel and R.  
This project focuses on data cleaning, filtering, and identifying patterns to generate insights that support real estate decision-making.

## Tools Used
- Excel (data filtering and preprocessing)
- R (data analysis and exploration)

## Project Overview
This project analyzes delinquent tax data to identify trends, patterns, and potential opportunities within the dataset.  
Certain proprietary filtering methods have been intentionally omitted.

## Status
Work in progress – additional analysis and refinements will be added.
